Description
____________
Features
This mod adds stables to the Imperial settlements.

Each stables lets you: 
- Leave a creature companion. Activate the hay to "park" your pet, activate your pet again to make it follow you.
Note: pet recall items still work
- Sleep on a haystack, hidden from the guards and the upstanding citizens
- Find small loot lost in haystacks.
- Cloud storage compatibility.
